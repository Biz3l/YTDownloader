## Description

YTMP3Downloader is an Electron + React-based desktop app for downloading YouTube videos as MP3 (or other formats supported by the downloader backend). It's designed to be simple, fast, and easy to use.

## Features

- Download YouTube videos by URL
- Convert video to MP3 audio
- Clean, minimal UI with theme support
- Built with modern React workflows

## Quick Start

### Prerequisites

- Node.js 18+ (or latest LTS)
- npm or yarn

### Install dependencies

```bash
cd /home/biz3l/Projects/YTMP3Downloader
npm install
# or yarn install
```

### Run in development

```bash
npm run dev
# or yarn dev
```

### Build production app

```bash
npm run make
# or yarn make
```
